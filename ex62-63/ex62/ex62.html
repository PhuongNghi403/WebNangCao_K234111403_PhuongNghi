Exercise 62 — Session Programming

Minh họa Session counter với Express-Session

📌 Cách hoạt động

Server dùng express-session. Mỗi lần gọi GET /contact, server đọc req.session.visited, tăng lên 1 và trả về kết quả. Reload trang vẫn giữ giá trị vì Session được lưu phía server, gắn với Session ID trong cookie. 

app.get("/contact", cors(), (req, res) => {
  if (req.session.visited != null) {
    req.session.visited++;
    res.send("You visited this page "
      + req.session.visited + " times");
  } else {
    req.session.visited = 1;
    res.send("Welcome to this page for the first time!");
  }
});
